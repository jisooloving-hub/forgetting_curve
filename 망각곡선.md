# 앱 아이디어

## 1. 복습 타이밍 자동 추천 앱

**개요:** 바쁜 현대 학생, 성인들을 위해 망각곡선 기반으로 "언제 다시 공부해야 하는지" 알려주는 앱

### 주요 기능
- 공부 내용 직접 입력 및 노트 작성
- 이해도 테스트를 통해 복습 간격 자동 조정 (두 가지 방식 중 선택)
  - **자가 체크** — 사용자가 직접 잘됨 / 안됨 선택
  - **AI 테스트** — 노트 내용 기반으로 AI가 문제 자동 생성, 풀이 결과로 이해도 판단
  - 잘됐으면 3일 후 복습 알림 (사용자가 시간 정할 수 있음)
  - 안됐으면 1일 후 복습 알림 (사용자가 시간 정할 수 있음)
- 복습 일정 캘린더 자동 등록

- 스트릭 — 며칠 연속으로 복습했는지 표시 (복습을 하루라도 빠지면 스트릭 초기화)

### 해야 할 것
- 망각곡선 가설 조사

## 2. 공부 동기부여 SNS

**개요:** 다른 사람들이 공부한 내용도 함께 보면서 본인도 공부한 내용을 인증해서 계속 공부하고 싶게 만들고, 동기부여 하는 시스템

### 핵심 아이디어
- 사람은 자기가 잘한 내용이나, 오늘 열심히 한 내용을 다른 사람에게 보여주고 싶어한다. 그걸 이용.

### 주요 기능
- 앱 내에서 타이머를 한 번도 시작하지 않았다면 사진 올릴 수 없음
- 시간을 정해두고 싶지 않다면 무제한 타이머 가능 (단, 최소 공부 시간 기준 필요)
- 사진은 갤러리 업로드 불가 — 공부 완료 후 바로 촬영만 허용 (책상, 플래너 사진 등)
- 피드에 사진 + 한 마디 첨부 가능
- 댓글 및 다양한 반응 옵션 (단순 좋아요 아닌 여러 옵션)
